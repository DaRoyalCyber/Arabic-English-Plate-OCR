 License Plate Recognition System (OCR) 🚗🔍

This project represents a professional implementation of Optical Character Recognition (OCR) and Computer Vision techniques to automate the detection and recognition of vehicle license plates in both Arabic and English.

 Project Overview

This project was developed as a graduation project and is designed to handle the complexity of different fonts and license plate styles used in Saudi Arabia.

 Technical Highlights

* Preprocessing: Noise reduction and image enhancement to improve character clarity.
* Localization: Detecting the exact license plate region within a full vehicle image.
* Segmentation: Splitting the plate into individual characters for recognition.
* Dual-Language Engine: Supporting both Arabic and English character recognition within the same system.

 Tech Stack

* Field: Computer Vision / Artificial Intelligence
* Key Concepts: OCR Algorithms, Image Processing, Pattern Recognition
